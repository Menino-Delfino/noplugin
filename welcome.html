How NoPlugin works

NoPlugin searches web pages for embedded plugin objects. If the file type is supported by your web browser (MP4, WAV, MP3, etc.), NoPlugin will play it directly in the web page. You can see a demo here.

If the file cannot be played by your web browser, NoPlugin can still download the file so you can open it with your computer's media player. VLC Media Player is recommended for opening these files, since it will play just about every video and audio file in existence.
