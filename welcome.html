Getting Started (read this!)

NoPlugin searches web pages for embedded plugin objects. If the file type is supported by your web browser (MP4, WAV, MP3, etc), NoPlugin will play it directly in the web page.

If the file cannot be played by your web browser, NoPlugin can still download the file so you can open it with your computer's media player. VLC Media Player is recommended for opening these files, because VLC can open almost any video or audio type (and it's free!).

Download VLC Media Player

Please do not leave NoPlugin a one-star review because it can't play a specific file type in your browser. That is a limitation of web browsers that cannot be fixed.
