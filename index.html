Sample videos

NoPlugin is able to play some plugin content in the web browser, as long as the plugin is trying to load an MP4, MP3, M4A, or WAV file. The below embed uses the QuickTime plugin to play MP4 video.

Here's another video, this time using the Windows Media Player plugin. NoPlugin can't play this in modern browsers.

If you have NoPlugin, all you have to do is click the 'Show Content' button, and the video will start to play in your browser. Easy!
